# Changelog

All notable changes to the Antifraud Medicine QR platform are documented in this file.

## [2.6.0] (2026-04-10)

### Initial Release

* public release of Antifraud Medicine QR under this repository
* FastAPI API endpoints for encode, decode, verify, and health checks
* ChaCha20-Poly1305 encryption with PBKDF2 key derivation
* QR generation with optional logo embedding
* local issuance registry with signature-based authenticity checks
